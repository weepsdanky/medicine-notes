## Anatomy — 结构

<!-- Crash Course A&P EP25–27, HMX Physiology -->

### 心脏

- 四个腔室：左心房（LA）、左心室（LV）、右心房（RA）、右心室（RV）
- 左心室壁最厚 → 负责把血泵到全身（体循环），压力最高
- 右心室 → 泵血到肺（肺循环），压力低

### 瓣膜（防止血液倒流）

| 瓣膜 | 位置 | 作用 |
|------|------|------|
| 二尖瓣 Mitral valve | 左房→左室 | 防止血液从左室流回左房 |
| 三尖瓣 Tricuspid valve | 右房→右室 | 防止血液从右室流回右房 |
| 主动脉瓣 Aortic valve | 左室→主动脉 | 防止主动脉血流回心脏 |
| 肺动脉瓣 Pulmonary valve | 右室→肺动脉 | 防止肺动脉血流回心脏 |

### 血液循环路径

```
体循环：左心室 → 主动脉 → 全身组织（送氧） → 上/下腔静脉 → 右心房
肺循环：右心室 → 肺动脉 → 肺（换气） → 肺静脉 → 左心房
```

## Physiology — 机制

<!-- HMX Physiology -->

### 心脏输出量（Cardiac Output）

```
CO = HR × SV
心输出量 = 心率 × 每搏输出量
```

- 正常值：约 5 L/min（安静时）
- 运动时可升至 20–25 L/min

### 血压（Blood Pressure）

```
BP = CO × TPR
血压 = 心输出量 × 外周血管阻力
```

- 收缩压（Systolic）：心脏收缩时的峰值压力
- 舒张压（Diastolic）：心脏舒张时的最低压力
- 正常值：<120/80 mmHg

### 心电图基础（ECG/EKG）

| 波形 | 含义 |
|------|------|
| P波 | 心房除极（心房收缩） |
| QRS波群 | 心室除极（心室收缩） |
| T波 | 心室复极（心室舒张） |

## Common Diseases — 常见病

### 高血压 Hypertension
- 定义：持续 BP >140/90 mmHg（中国标准）
- 机制：RAAS 过度激活 → 血管收缩 + 钠水潴留 → BP 升高
- 危险因素：高盐饮食、肥胖、压力、遗传、肾病
- 后果（靶器官损害）：心脏（心肌肥厚/心衰）、肾脏（肾功能不全）、脑（中风）、眼底

### 冠心病 Coronary Artery Disease (CAD)
- 机制：冠状动脉粥样硬化 → 管腔狭窄 → 心肌供血不足（缺血）
- 症状：胸闷、胸痛（心绞痛 Angina）、放射至左肩/下颌
- 急性发作：斑块破裂 → 血栓形成 → 完全堵塞 → 心肌梗死（MI）

### 心力衰竭 Heart Failure (HF)
- 心脏泵血能力下降，无法满足全身需求
- 左心衰：肺水肿（呼吸困难、端坐呼吸）
- 右心衰：体循环淤血（下肢水肿、颈静脉怒张）
- 分类：HFrEF（射血分数降低）vs HFpEF（射血分数保留）

## Pharmacology — 相关药物

| 药物类别 | 英文 | 作用 | 常见药名 |
|----------|------|------|---------|
| ACEI | ACE Inhibitors | 扩血管，降压，保护心肾 | 卡托普利、依那普利 |
| ARB | ARBs | 同 ACEI，副作用少 | 氯沙坦、缬沙坦 |
| β受体阻滞剂 | Beta-blockers | 减慢心率，降压，治心衰 | 美托洛尔、比索洛尔 |
| 他汀类 | Statins | 降低 LDL，稳定斑块 | 阿托伐他汀、瑞舒伐他汀 |
| 利尿剂 | Diuretics | 排钠排水，降低前负荷 | 氢氯噻嗪（噻嗪类）、呋塞米（袢利尿剂） |
| 硝酸酯类 | Nitrates | 扩张冠脉，缓解心绞痛 | 硝酸甘油 |
| 阿司匹林 | Aspirin | 抗血小板，防止血栓 | 阿司匹林肠溶片 |

## Lab Values — 相关检验指标

| 指标 | 英文 | 正常值 | 临床意义 |
|------|------|--------|---------|
| 心肌肌钙蛋白 | Troponin (I/T) | 极低（各实验室标准不同） | 升高 = 心肌损伤，心梗指标 |
| CK-MB | CK-MB | <25 U/L | 心肌损伤标志物 |

| 甘油三酯 | TG | <1.7 mmol/L | 代谢综合征指标 |

## In the Hospital — 医院里会听到什么

### 常见术语

| 术语 | 含义 |
|------|------|
| 心肌缺血 | 心肌供血不足，但还没坏死 |
| 心肌梗死（心梗）| MI，心肌因缺血坏死 |
| 心绞痛 | Angina，胸痛但心肌未坏死 |
| 射血分数（EF）| 心室每次收缩射出血量的百分比，正常 55–70% |
| 心律失常 | Arrhythmia，心跳节律不正常 |
| 房颤 | Atrial fibrillation (AFib)，心房乱颤，最常见的心律失常 |
| 心超 | 超声心动图（Echocardiogram），看心脏结构和功能 |
| 冠造 | 冠状动脉造影，确认堵在哪里 |
| 支架 | 冠状动脉介入治疗（PCI），撑开堵塞的冠脉 |

### 医生常问的问题（及为什么问）

- "有没有胸闷、胸痛？" → 筛查心绞痛/心梗
- "爬几层楼开始喘？" → 评估心功能（心衰分级）
- "晚上能平躺吗？" → 端坐呼吸是左心衰典型症状
- "脚肿不肿？" → 右心衰导致体循环淤血
- "家里人有没有心脏病？" → 遗传风险
