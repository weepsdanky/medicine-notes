# Anatomy & Physiology — 基础概论

Crash Course A&P 前几集（EP01–EP06 左右）的内容。
跨系统的基础概念，不属于某一个器官系统。

---

## 解剖学 vs 生理学

- **Anatomy（解剖学）**：研究结构——长什么样，在哪里，叫什么名字
- **Physiology（生理学）**：研究功能——怎么工作，为什么这么工作
- 两者密不可分：结构决定功能（心脏的形状决定它怎么泵血）

## 人体组织层级

从小到大：

```
化学层 → 细胞层 → 组织层 → 器官层 → 器官系统层 → 整体
```

| 层级 | 英文 | 例子 |
|------|------|------|
| 细胞 | Cell | 心肌细胞、神经元 |
| 组织 | Tissue | 肌肉组织、上皮组织 |
| 器官 | Organ | 心脏、肾脏 |
| 器官系统 | Organ system | 心血管系统、神经系统 |

---

## 四种基本组织类型

| 组织类型 | 英文 | 功能 | 例子 |
|----------|------|------|------|
| 上皮组织 | Epithelial tissue | 覆盖表面，分泌，保护 | 皮肤、消化道内壁 |
| 结缔组织 | Connective tissue | 支撑，连接，储存 | 骨骼、脂肪、血液 |
| 肌肉组织 | Muscle tissue | 收缩产生运动 | 心肌、骨骼肌、平滑肌 |
| 神经组织 | Nervous tissue | 传递电信号 | 脑、脊髓、神经 |

## 稳态（Homeostasis）

身体最核心的运作原则：**维持内环境稳定**。

- 体温、血糖、血压、pH 都被精确调控
- 机制：**负反馈**（Negative feedback）——偏离目标值 → 触发纠正 → 回到目标值
- 例子：血糖升高 → 胰岛素分泌 → 细胞摄取葡萄糖 → 血糖降回正常

**正反馈**（Positive feedback）少见，用于需要"加速到底"的场景：
- 例子：分娩宫缩（越缩越强）、血液凝固

## 解剖方位术语

| 术语 | 含义 | 中文 |
|------|------|------|
| Anterior / Posterior | 前 / 后 | 腹侧 / 背侧 |
| Superior / Inferior | 上 / 下 | 靠头 / 靠脚 |
| Medial / Lateral | 内侧 / 外侧 | 靠中线 / 远离中线 |
| Proximal / Distal | 近端 / 远端 | 靠躯干 / 远离躯干（用于四肢） |
| Superficial / Deep | 浅 / 深 | 靠表面 / 靠内部 |

## 体腔（Body Cavities）

| 体腔 | 内容 |
|------|------|
| 颅腔 Cranial cavity | 大脑 |
| 脊椎管 Vertebral canal | 脊髓 |
| 胸腔 Thoracic cavity | 心脏、肺（被纵隔分隔） |
| 腹腔 Abdominal cavity | 消化器官、肝、脾、肾 |
| 盆腔 Pelvic cavity | 膀胱、生殖器官、直肠 |
