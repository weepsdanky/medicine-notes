# medicine-notes

系统学习医学的笔记。从基础医学出发，逐步深入感兴趣的方向。

## 学习目标

不追求临床执照，目标是系统理解人体运作机制，建立医学知识框架。

## 学习路线（规划中）

```
基础医学:
  ├── 解剖学（系统解剖 — 心血管、神经、肌骨）
  ├── 生理学（各系统功能机制）
  ├── 生物化学（代谢、蛋白质、基因表达）
  └── 病理学（疾病机制入门）

进阶方向（待确认）:
  ├── 神经科学
  ├── 免疫学
  └── 药理学基础
```

## 目录结构

```
anatomy/       ← 解剖学笔记
physiology/    ← 生理学笔记
biochemistry/  ← 生化笔记
pathology/     ← 病理学笔记
neuro/         ← 神经科学（如果深入）

```
