# medicine-notes

系统学习医学的笔记。目标：在医院（中/外）能听懂术语、理解医生在说什么。
不追求临床执照，不准备考试。定位：**患者视角的理解**。

---

## 目录结构

```
foundations/                ← 跨系统的基础学科
  anatomy-physiology-basics.md   ← 细胞、组织、器官系统概论（Crash Course 前几集）
  biochemistry.md           ← HMX Biochemistry
  genetics.md               ← HMX Genetics
  pharmacology.md           ← HMX Pharmacology

systems/                    ← 按器官系统（解剖+生理+常见病+药物全放一起）
  cardiovascular.md
  respiratory.md
  nervous-system.md
  renal.md
  endocrine.md
  immune.md
  gastrointestinal.md
  musculoskeletal.md

refs.md                     ← 课程资源与学习路线
```

---

## 每个 systems/ 文件的结构

```markdown
# [System Name]

## Anatomy — 结构
（这个系统长什么样，各部分叫什么）

## Physiology — 机制
（正常情况下是怎么工作的）

## Common Diseases — 常见病
（常见的病，机制是什么，为什么会发生）

## Pharmacology — 相关药物
（医生常开的药，是什么类，为什么用它）

## Lab Values — 相关检验指标
（拿到化验单时哪些数字跟这个系统有关，正常值是多少）

## In the Hospital — 医院里会听到什么
（门诊/住院常见术语、医生的问法、常见诊断名称）
```

## 记笔记的方式

**看 Crash Course A&P：**
- 看完一集，打开对应系统的 .md 文件，写下这集讲的关键点
- 只写"你觉得重要的/你之前不知道的"，不要转录
- 格式不重要，先写上去，之后再整理

**看 HMX / Khan Academy：**
- 按照 systems/ 的模板往对应 section 里填
- 来源标注在行尾：`<!-- HMX Physiology W2 -->`

**遇到医院里的术语：**
- 先查 refs.md 里的术语速查方法
- 然后往对应系统的 "In the Hospital" 一节补充
